# SCT_DS_1
Data visualization project using Python, Pandas, and Matplotlib for SkillCraft Technology Internship Task 01.
SkillCraft Technology Internship - Task 01

Objective:
Create a bar chart or histogram to visualize the distribution of categorical or continuous variables.

Dataset Used-
Titanic Dataset

Tools Used-
- Python
- Pandas
- Matplotlib
- Google Colab

Visualizations-
1. Histogram for Age Distribution
2. Bar Chart for Gender Distribution

Conclusion-
The project visualizes passenger age and gender distributions using basic data visualization techniques.
